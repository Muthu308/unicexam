Attendance System

Choose your role

Pick where you'd like to go. Each view talks to the same Hasura-backed register, so a class added by the Admin shows up instantly for Faculty, and attendance marked by Faculty shows up instantly for the Student.

How it fits together

  • Admin creates a class: session, subject, topic, faculty name, and the roster of student IDs.
  • Faculty opens a class and marks each student Present, Absent, or gives a Reason.
  • Student looks up their attendance history by student ID.
